## Как запустить приложение

1. База данных:
```powershell
docker compose up -d db
```
  
2. Приложение:
```powershell
uv sync
uv run uvicorn main:app --reload --host 0.0.0.0 --port 8888
```

## Как открыть и использовать приложение

1. После запуска сервера откройте в браузере:
   **http://127.0.0.1:8888**

2. Интерфейс приложения:

   - **Форма вверху страницы** — для создания новой задачи:
     - Введите заголовок (обязательно)
     - При желании добавьте описание
     - Нажмите **"Добавить задачу"**

   - **Список задач ниже**:
     - Каждая задача отображается в отдельной карточке
     - Справа от задачи две кнопки:
       - **В процессе** / **Выполнено** — переключает статус задачи (текст зачёркивается при выполнении)
       - **Удалить** — удаляет задачу (с подтверждением)

   Все действия (добавление, изменение статуса, удаление) происходят **мгновенно без перезагрузки страницы** благодаря HTMX.

Готово! Теперь вы можете полноценно управлять списком задач.
